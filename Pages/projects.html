Projects

I truly enjoy developing games and I try my best to gain as much knowledge as possible in as many related fields. Therefore, in order to do so, I create games in which I cover design, art and programming.
Different games rise different challenges and so I am pushed to experience and document myself in order to reach the game's end goal.
On this page I feature some of my most polished projects I've done.

Floors of Ages

Period of creation: November 2016 - Present
Platform: PC
Engine: Unity Engine (C#)
Development blog: Game dev progression
Summary: "Floors of Ages" is a rogue-like in which you age instead of losing health upon taking damage. The main character can get older, rejuvenate and back again. The age influences three main stats which again influence other minor bonuses.
The story revolves around a man who's trapped underground, deep into the Earth. In order to reach the above he has to climb a series of floors. Every new floor brings new challanges as well as new powers.
Most of the sessions concludes with the death of the main character but its names will always be remembered. The game features an online leader-board that holds the top progresses.
A lot more about the project can be discovered by following the development blog.

Wander

Release date: January 2016
Platform: Android Phone
Engine: Unity Engine (C#)
Game's page: Google Play
Summary: "Wander" is an endless, progressively harder, top-down runner where the goal is to get as far as possible without touching any of the blocks. Dodge the deadly walls and aim to get the highest score.
"Wander" is my first game created for a smart-phone device. It is also the most popular with over 800 downloads.
The project taught me to pay great attention to optimisation and user feedback. A lot of changes were brought to the game based on external feedback.

Aerolite Defense

Release date: January 2016
Platform: PC
Engine: Unity Engine (C#)
Game's page: Game Jolt
Summary: Aerolite Defense is an arcade defense type game in which you have to protect your planet by matching the shield colour with the meteorites'.
The game was created over a period of a week as a challenge for myself.
By doing so I've learned how to iterate fast, how to focus on major elements and keep the game concise and within the scope. By keeping the game simple and focused I managed to create a complete experience which I was able to post online.

Pixieful

Release date: December 2015
Platform: Android Phone
Engine: Unity Engine (C#)
Game's page: Google Play
Summary: Pixieful is an arcade endless-runner-like game in which you have to stay in the safe zone in order to get your score as high as possible and dominate the leader-board.
The game features an online leader-board, six different maps and eight aesthetically different character.
Perhaps the most complex game I've created for the mobile platform until that time.

Spell Bash

Release date: December 2015
Platform: Arcade machine
Engine: Unity Engine (C#)
Source: GitHub scripts repository
Summary: "Spell Bash" is a PVP local arena game in which you have to manage and properly use your arsenal of spells in order to defeat your enemy.
There are six spells to use, equally available for both players. These abilities use mana and each have particular cooldowns. It is up to the player to properly use them at the right moment in the right sequence in order to triumph over the adversary.
The game is available at university on an arcade machine.

Slide Effect

Release date: February 2016
Platform: Android Phone
Engine: Unity Engine (C#)
Source: Google Play
Summary: "Slide Effect" is an arcade game in which you have to dodge the falling obstacles for as long as you can.
My brother told me at some point that he would like to be part of a project which he can play on the go. So, we teamed-up to create "Slide Effect". The game idea is entirely his therefore all the credits go to him.
The project was developed in a very short amount of time.
The game can be downloaded for free from Google Play.
