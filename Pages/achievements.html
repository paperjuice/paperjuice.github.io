Global Games Jam 2017

During this Global Games Jam I was part of a team of four: one project manager, one artist, one programmer and me, programmer.
Our project was aimed for HTC Vive. The game is a first person puzzle in which you have move object to certain position in order to meet the requirement and progress further.
The objects could be moved horizontally using the controllers and in order to lift the objects the player had to use voice input (the higher the pitch the stronger the push power). After a certain pitch level, the object is tossed forward.
The game can be found by following this link.
I had the following duties:

Program the object movement both volume and controller driven

Work on particle effects

Add post processing effects
